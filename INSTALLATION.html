🚀 Installation Guide - Learn Urdu Quarto Project

Complete step-by-step instructions for setting up your Urdu learning platform.

📋 Prerequisites Checklist

Before starting, ensure you have:

Step 1: Install R (10 minutes)

Windows

  1. Go to https://cran.r-project.org/bin/windows/base/
  2. Click “Download R-4.x.x for Windows”
  3. Run the installer
  4. Follow default installation options
  5. Verify: Open CMD and type R --version

Mac

  1. Go to https://cran.r-project.org/bin/macosx/
  2. Download R-4.x.x.pkg
  3. Double-click to install
  4. Verify in Terminal: R --version

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install r-base
R --version

Step 2: Install RStudio (10 minutes)

  1. Download RStudio Desktop (Free)
    • Visit: https://www.rstudio.com/products/rstudio/download/
    • Click “Download RStudio Desktop”
    • Choose your operating system
  2. Install
    • Run the installer
    • Follow default options
    • Launch RStudio to verify

Step 3: Install Quarto (5 minutes)

  1. Download Quarto

    • Visit: https://quarto.org/docs/get-started/
    • Click download for your OS
    • Version 1.3+ required

  2. Install

    • Run installer
    • Follow prompts
    • Restart RStudio after installation

  3. Verify Installation

    quarto --version

    Should show version 1.3.0 or higher

Step 4: Install R Packages (15 minutes)

Open RStudio and run:

# Essential packages
install.packages(c(
  "quarto",
  "dplyr",
  "ggplot2",
  "DT",
  "knitr",
  "kableExtra",
  "htmltools",
  "rmarkdown",
  "jsonlite"
))

# Verify installation
library(quarto)
library(dplyr)
library(ggplot2)

If you get errors:

# Try installing one at a time
install.packages("dplyr")
# Then the next one, etc.

Step 5: Download Project Files (5 minutes)

Option A: Direct Download

  1. Download all files I provided
  2. Create folder: learn-urdu-quarto/
  3. Organize files according to structure shown in README

Option B: Clone from GitHub (if available)

git clone https://github.com/yourusername/learn-urdu-quarto.git
cd learn-urdu-quarto

Step 6: Set Up Project Structure (10 minutes)

Create this exact folder structure:

learn-urdu-quarto/
├── _quarto.yml
├── index.qmd
├── alphabet.qmd
├── vocabulary.qmd
├── grammar.qmd
├── conversations.qmd
├── quizzes.qmd
├── writing.qmd
├── challenges.qmd
├── progress.qmd
├── study-buddy.qmd
├── README.md
├── data/
│   ├── vocabulary.csv
│   └── (other data files)
├── audio/
│   └── (audio files - optional)
├── js/
│   ├── audio-player.js
│   ├── quiz-engine.js
│   ├── writing-practice.js
│   └── progress-tracker.js
├── css/
│   └── custom.css
└── R/
    └── (R scripts - optional)

Quick Setup Script:

Windows (PowerShell):

# Navigate to your desired location
cd C:\Users\YourName\Documents

# Create structure
mkdir learn-urdu-quarto
cd learn-urdu-quarto
mkdir data, audio, js, css, R

Mac/Linux (Terminal):

# Navigate to your desired location
cd ~/Documents

# Create structure
mkdir -p learn-urdu-quarto/{data,audio,js,css,R}
cd learn-urdu-quarto

Step 7: Add Your Files (10 minutes)

  1. Copy all .qmd files to project root
  2. Copy data files to data/ folder
  3. Copy JavaScript files to js/ folder
  4. Copy CSS file to css/ folder

Verify File Structure:

# In RStudio, check files
list.files()  # Should show .qmd files
list.files("data/")  # Should show .csv files
list.files("js/")  # Should show .js files

Step 8: First Build (5 minutes)

  1. Open RStudio

  2. Open Project:

    • File → Open Project
    • Navigate to learn-urdu-quarto/
    • Select learn-urdu-quarto.Rproj (if exists)
    • Or: File → New Project → Existing Directory

  3. Render Site:

    quarto::quarto_render()

  4. Preview Site:

    quarto::quarto_preview()

    Should open in browser!

Step 9: Add Audio Files (Optional - 30 minutes)

Option 1: Text-to-Speech (Easiest)

  • Skip audio files completely
  • System uses browser TTS automatically
  • Works immediately

Option 2: Download Audio

  1. Visit Forvo.com
  2. Search for Urdu words
  3. Download pronunciations
  4. Save to audio/ folder with exact filenames from CSV

Option 3: Record Your Own

  1. Use Audacity (free)
  2. Record pronunciations
  3. Export as MP3
  4. Match filenames in vocabulary.csv

Audio File Naming:

audio/
├── assalam.mp3
├── shukriya.mp3
├── haan.mp3
├── nahi.mp3
└── ...

Step 10: Test Everything (10 minutes)

Test Checklist:

Common Issues:

Urdu text looks wrong:

# Check _quarto.yml has:
include-in-header:
  - text: |
      <link href="https://fonts.googleapis.com/css2?family=Noto+Nastaliq+Urdu:wght@400;700&display=swap" rel="stylesheet">

Tables don’t show:

# Install missing package
install.packages("DT")
install.packages("kableExtra")

Audio doesn’t work: - Normal! Just means files not added yet - Will use text-to-speech fallback

Step 11: Deploy to Web (20 minutes)

Option A: GitHub Pages (Recommended)

  1. Create GitHub Account (if needed)

    • Go to github.com
    • Sign up for free

  2. Create Repository

    • Click “New repository”
    • Name: learn-urdu
    • Public
    • Initialize with README

  3. Upload Files

    # In your project folder
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/learn-urdu.git
git push -u origin main

  4. Enable GitHub Pages

    • Repository → Settings → Pages
    • Source: Deploy from branch
    • Branch: main
    • Folder: /docs
    • Save

  5. Build and Deploy

    # Render to docs folder
quarto::quarto_render()
    # Push to GitHub
git add docs/
git commit -m "Deploy website"
git push

  6. Your Site is Live!

    • URL: https://yourusername.github.io/learn-urdu/
    • Takes 2-5 minutes to go live

Option B: Add to myaseen208.com

If you already have myaseen208.com:

  1. Build Site

    quarto::quarto_render()

  2. Copy docs folder to your website

    • Rename to: learn-urdu/
    • Upload to: myaseen208.com/learn-urdu/

  3. Update your main site Add link:

    <a href="/learn-urdu/">Learn Urdu</a>

Step 12: Maintain and Update (Ongoing)

Adding New Vocabulary:

  1. Edit data/vocabulary.csv
  2. Add row: category,difficulty,urdu,roman,english,audio.mp3
  3. Run: quarto::quarto_render()
  4. Push to GitHub

Adding New Quizzes:

  1. Edit quizzes.qmd
  2. Add questions to quizDatabase object
  3. Render and deploy

Customizing Colors:

  1. Edit css/custom.css
  2. Change color values
  3. Render and deploy

🎉 You’re Done!

Your Urdu learning platform is now live!

📞 Getting Help

If Something Doesn’t Work:

  1. Check Console:

    # In RStudio, check for errors
quarto::quarto_check()

  2. Verify Packages:

    # List installed packages
installed.packages()[,1]

  3. Update Everything:

    # Update R packages
update.packages(ask = FALSE)

# Update Quarto
quarto::quarto_update()

  4. Common Fixes:

    # Clear cache
quarto::quarto_render(cache_refresh = TRUE)

# Restart R Session
# Session → Restart R

Still Stuck?

  • Check README.md for troubleshooting
  • Search error messages on Google
  • Post issue on GitHub
  • Email: contact@myaseen208.com

# Re-render
quarto::quarto_render()
